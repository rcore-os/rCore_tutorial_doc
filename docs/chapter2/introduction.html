第二章：最小化内核

第二章：最小化内核

本章概要

在上一章中，我们移除了程序中所有对于已有操作系统的依赖。但是我们的内核开发仍然需要依赖硬件平台。现在让我们来看一看怎样才能让我们的内核在硬件平台上跑起来。

本章你将会学到：

  • 使用 目标三元组 描述目标平台
  • 使用 链接脚本 描述内存布局
  • 进行 交叉编译 生成可执行文件，进而生成内核镜像
  • 使用 OpenSBI 作为 bootloader 加载内核镜像，并使用 Qemu 进行模拟
  • 使用 OpenSBI 提供的服务，在屏幕上格式化打印字符串用于以后调试
